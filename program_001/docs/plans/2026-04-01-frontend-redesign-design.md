# 云端编程 (Cloud Programming) - 落地页 UI/UX 彻底重构设计方案

## 1. 概述与核心理念 (Overview)
目标是对现有的 `/web_cs/program_001` 落地页进行重构，将其原本“深色/冷科技风”的基调彻底改为 **“活泼教育 / 游戏化 (Gamified & Educational)”** 风格，提升学习平台的亲和力。核心目标是通过更加友好的视觉组件和动效，降低编程新手的入门心理门槛。

## 2. 视觉规范 (Visual & Typography)
*   **字体系列 (Typography)**：引入 `Nunito` 或 `Quicksand`（谷歌字体 CDN）。用大圆角、笔触圆润的字体赋予整个页面轻松、友好的性格。
*   **色彩空间 (Color Palette)**：
    *   主色调 (Primary)：阳光橙/珊瑚红 (例如：`#FF7E67`, `#FF5470`)
    *   辅助色 (Accent)：薄荷绿 / 清爽蓝 (例如：`#2DE1C2`, `#48BB78`, `#2D8EFF`)
    *   背景 (Background)：主背景使用纯白及极浅的低饱和度暖灰（`#FDFDFD`, `#FFF9F8`），利用留白增强呼吸感。
    *   文字 (Text)：深蓝色或蓝灰，避免死板的纯黑（`#1E293B`, `#334155`）。
*   **组件形态 (Shapes & Forms)**：
    *   大量采用 **Neo-Brutalism（新拟态/粗边框块状风格）** 或 Duolingo 式的 **厚度 3D 风格**。
    *   按钮底部带有较深的彩色投影，模拟积木般的厚度，触发按下状态（`:active`）时阴影减小，按钮位置下移，形成真实的物理点击反馈。
    *   卡片增加较大的 `border-radius: 24px;`。

## 3. 核心版块重构方案 (Sections Redesign)

### 3.1 Hero 首屏
*   **背景**：移除原有的星空/深色渐变背景，替换为由 SVG 生成的多彩波浪形 (Wavy) 背景。
*   **装饰**：在主内容周围增加漂浮的 Emoji 风格图标/插画（如 🚀 小火箭, 💻 积木块代码, ☁️ 云朵）。
*   **CTA**：巨型的、具有呼吸动画（Bouncing）的“立即体验”3D 按钮。

### 3.2 云端实验室版块 (Lab Section)
*   **视觉隐喻**：将暗黑、死板的代码编辑器 UI 重制为一个“彩色玩具盒”。
*   **UI 细节**：
    *   编辑器的主题由暗变亮（浅色底 + 彩色高亮关键词），比如淡黄色的背景加上橙/绿的高亮字。
    *   浏览器外壳采用柔和的色块，三大控制按钮（红黄绿）更加圆润可爱。

### 3.3 双屏学习区 (Dual Screen)
*   **布局重构**：打破生硬的两个长方形方块，将视频区与实操区设计为通过曲线/虚线连接的两个气泡形卡片。
*   **卡片风格**：增加悬浮深度，突出“看视频”与“动手敲”两者间的联动性。

### 3.4 课程特色版块 (Features Cards)
*   **卡片设计**：3D 悬浮卡片式设计。每张卡片底部有深一号的同色块充当厚度阴影。
*   **交互**：鼠标悬浮时，卡片呈现上浮的弹性动效。
*   **图标**：放弃细线条的 Bootstrap Icon，改为基于 CSS 或大圆角的粗体大图标（或者 Emoji/渐变大圆底背景图标）。

## 4. 动效细节 (Micro-Interactions)
*   所有按钮、卡片的 Hover 使用 `cubic-bezier(0.34, 1.56, 0.64, 1)`（弹簧/果冻效果）。
*   向下滚动页面时，区块内容应用由下至上的渐隐弹出效果（Pop-in animation，结合 `IntersectionObserver` 实现）。

## 5. 开发建议 (Tech Stack)
*   继续使用 HTML5 + Bootstrap 5 + 纯 CSS，但在 `style.css` 中大面积重写覆盖变量和类。
*   减少对外部过重资源的依赖，SVG 全部内联或使用纯 CSS 绘制积木/3D 效果。
